상품권 판매 사이트 구축관련
구현해야될 메뉴 및 화면
(1). 카테고리 - 갤러리아, 신세계, 롯데
(2). 각카테고리별 상품리스트 -> 상세화면
(3). 메인화면 -> 상세화면 (메인화면은 인기상품, 전체상품)
(4). 주문/결제 -> 장바구니,주문/결제,주문완료
(5). 로그인 -> 아이디, 패스워드 찾기
(6). 회원가입(실명인증,약관동의,회원정보입력,가입완료)
(7). 고객센터 
- 공지사항
- 문의사항
- 나의쇼핑-(장바구니,주문/배송조회,취소/반품/교환 조회,환불/입금내역)
- 회원정보-(나의배송지관리)
- 이용약관-(서비스이용약관,전자금융거래약관,개인정보수집,이메일수집)
(8). 기본적으로 기존 db sound와 동일하게 일단 개발
	추후에 기능 및 화면이 변경될수도 있음.
	일단 기능 구현 먼저 진행합시다.




관리자 개발 방법
1. 개발해야할 기존 admin/inc/left.jsp 
에서 해당 메뉴 url을 검색
giftcard/admin/inc/left.jsp 에 복사하기
2. 검색결과에서 해당 url호출하는 controller를 giftcard 팩키지 하위에 복사
3. sql파일은 /resource/sql-map-config.xml 파일에서 
복사한 sql 파일을 추가하고 기존 동일한 명의 sql명은 주석처리한다.
3. controller명과 service 명 메소드 접근 url, 리턴 jsp경로 등등 수정 
 (네이밍룰은 클래스명은 GiftCard)
 (접근 url,리턴 jsp명은 /giftcard)
 (db 테이블을 만들어야할경우는 SC_테이블명)
4. controller 리턴 되는 jsp 복사하기
5. jsp 디자인 작업 및 로직구현하기 끝

메인화면도 동일하게 1번을 제외하고 
기존 url을 giftcard 패키지및 경로 수정 진행하면된다.
